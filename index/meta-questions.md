# Meta-Question Categories Index

Based on Meta-Question Oriented Numbering System v2.1

## Format

```
m[XX][YYY][ZZZZZ]
```
- XX: Meta-question category (01-15)
- YYY: Technical subcategory (001-999)
- ZZZZZ: Sequence number
